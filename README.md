# Python URL Shortener

A simple URL shortener web app built using Python and Flask.

##  Features
- Convert long URLs into short links
- Redirect short URL to original URL
- Uses SQLite database
- Simple web interface

##  How to Run

1. Install Python
2. Install Flask:

pip install flask

3. Run app:

python app.py

4. Open in browser:

http://127.0.0.1:5000/

## Technologies Used
- Python
- Flask
- SQLite
